# Work Review v1.0.53

**發布日期**：2026-06-28

## 新增

- **刪除活動記錄**：時間線支援按單條 / 按日期 / 按時間段 / 按應用四種粒度刪除活動記錄，連帶刪除對應截圖與 OCR 文本；刪除後自動失效受影響日期的日報與小時摘要快取，統計與日報保持一致（下次查看時重新生成）。
- **介面視覺風格 A/B/C**：新增 Quiet Pro / Soft Layers / Compact Data 三套介面風格，每套帶預覽，經 `ui_visual_style` 配置持久化（Beta）。

## 修復

- **英文模式下中文殘留**（#111）：將繞過 i18n 的硬編碼 UI 文案（視窗按鈕、AI 測試報錯、桌寵 aria-label、圖表圖例等）提取為三語 key 並對齊；助手 starter prompt 改為按 locale 切換（含動態數據翻譯）；新增 `I18nEnglishLeak` 回歸測試守住 title/aria-label/placeholder 不含中文與三語 key 對齊。
- **自訂分類在日報時間分配裡遺失**（#109）：`normalize_category_key` 此前把任何非內建分類摺疊到 "other"，導致日報時間分配看不到自訂分類；改為聚合時保留自訂 key。
- **時間線「週期摘要」按鈕日期跳到今天**（#110）：現在攜帶當前查看日期；同時修復從 `location.hash` 讀取 `?date`（hash router 下 `location.search` 為空的潛在 bug，也影響桌寵繼續）。
- **Linux KDE Plasma Wayland 啟動崩潰**（#113）：上游 webkit2gtk/GTK 在 Wayland 下的協定錯誤（`Error 71 dispatching to Wayland display`，Tauri #10702），在 GTK 初始化前注入 `WEBKIT_DISABLE_DMABUF_RENDERER=1` 規避；README 文件化該問題與 `GDK_BACKEND=x11` 兜底。
- **響應式**：日報 Markdown 表格橫向捲動、長 URL/文本換行不再撐破欄；node gateway 設備表單與概覽瀏覽器卡片在窄視窗下正確 truncate。
- **更新器鏡像失效**：原 `ghproxy.cn` + `ghp.ci` 回傳 HTTP 000，換為 `gh-proxy.cn` + `gh-proxy.com`（驗證 HTTP 200）。
- **崩潰無記錄**：啟動安裝 panic hook，崩潰資訊（版本/訊息/位置/堆疊）寫入 `<data_dir>/crashes/crash-<ts>.log`（此前靜默退出無法診斷）。
- **AiTransform model 綁定**（曾被誤改為 `_model`）恢復。
- **自訂分類名在其他語言下被誤翻譯**：自訂（非內建）分類現在保留使用者定義的名字，不再被 `translateCategoryLabel` 誤覆蓋；內建分類仍按語言翻譯。影響時間線、概覽、每小時圖表的分類名顯示。
- **Linux 建構失敗**（`commands.rs` 拆分遺留）：`avatar.rs` 移到 `commands/` 子目錄後，`include_str!` 相對路徑深一層未更新（`../../` → `../../../`）；`system.rs` 跨模組呼叫 `avatar.rs` 的 `gnome_avatar_extension_*` 私有函式未補 `use`，且 `avatar.rs` 缺 `PathBuf` import。修復後 Linux CI 編譯通過。

## 優化

- **桌寵 GPU 佔用**：移除 `.scene-svg` 的 `will-change`（曾強制整場景常駐 GPU 合成層）+ 降低 drop-shadow 模糊半徑 18→12px（視覺近乎不變，減少逐幀 filter 開銷）。
- **語言切換展示**：當前語言改用深色背景標記（去掉 ✓）；README 當前語言加粗且不再自連結，「中文」改為「簡體中文」以消歧義。
- **程式碼品質**：移除約 981 行死碼（16 個未用函式及測試 helper）；6 處重複的隱私過濾函式收斂到 `core/privacy.rs` 單一來源；`commands.rs`（8517 行）按領域拆為 14 個子模組（行為不變，註冊路徑不變）；workspace 0 warning。
- **每小時圖表時長格式 locale 化**：座標軸刻度與圖例時長改用 `formatDurationLocalized` 跟隨當前語言（此前硬編碼 `Xh/Xm/Xs`）。
- **README 介面預覽擴充**：三語 README 新增工作流 GIF 與各介面截圖（概覽 / 時間線 / 時間線詳情 / 日報 / 助手 / 接入管理 / 小時總結 / 各設定頁），覆蓋完整工作流與配置面；配套截圖腳本與守護測試。

```bash
sudo xattr -rd com.apple.quarantine '/Applications/Work Review.app'
```
