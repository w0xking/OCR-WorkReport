# Work Review v1.0.54

**發布日期**：2026-07-02

## 修復

- **隱藏的日報區塊無法恢復**（#109）：`apply_preferences` 生成日報時不再過濾隱藏區塊，隱藏完全交給前端顯示層。在「管理段落」恢復後立即可見，無需重新生成。
- **托盤選單英文模式下顯示中文**（#115）：config 新增 `locale` 欄位，托盤選單文案三語映射，切換語言時即時刷新托盤。
- **助手回答不跟隨提問語言**（#116）：AI 增強 / 閒聊 / 無模型三條路徑現在都按使用者提問語言回答，英文提問→英文回答、中文提問→中文回答，不管工作記錄裡存的是什麼語言。
- **FastPath 分類名中文殘留**：`fast_answer` 分類分布按提問語言分流，英文模式不再夾中文分類名。
- **更新器兜底泄漏**：未識別的後端中文狀態串不再原樣返回，避免英文模式泄漏。
- **Linux 建構失敗**（1.0.53 遺留）：修復 `commands.rs` 拆分後 `include_str!` 路徑、跨模組引用、`PathBuf` import。
- **托盤命令編譯**：`set_app_locale` 改私有（避免巨集重導出衝突）+ `AppHandle` 所有權修復。
- **日報生成閉包語法**：修復 `apply_preferences` 閉包參數多餘 `>`。

## 優化

- **靜默駐留 GPU 佔用**（#118）：開機自啟 + 靜默駐留 + 輕量模式下，主視窗隱藏時暫停所有 CSS 動畫，WebView2 GPU 佔用從 7-30% 降到接近 0。
- **概覽日期欄**：刪除日期選擇欄下方的分隔橫線 + 收緊高度。日曆式選擇控制項實測後保留原 text input。
- **桌寵設定文案**：精簡桌寵相關描述；數據說明新增「會消耗少量 token」提示。

## macOS 安裝說明

首次啟動可能看到「已損壞」提示，執行：

```bash
sudo xattr -rd com.apple.quarantine '/Applications/Work Review.app'
```
