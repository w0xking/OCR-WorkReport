# Work Review v1.0.52

**發布日期**：2026-06-23

## 新增

- **AI 編排日報區塊**：日報改為 AI 依據當日工作模式決定區塊順序，每區塊支援固定（pin）/隱藏（hide），使用者偏好始終覆蓋 AI 順序；AI 排序快取到設定避免重複 LLM 呼叫；區塊按順序動態編號（一、二、…）。
- **主動式桌寵**：由模型自定開口節奏／內容／情緒（`next_check_minutes`），跟進卡片裁剪，卡片或氣泡顯示期間視窗拖曳兜底。
- **動態開場提示**：設定 AI 模型後，工作助手依工作情境動態生成開場提示（新增 `generate_text_with_model` 命令）。
- **小時活躍度多模式**：概覽支援按週／今日／日期範圍查看每小時應用分布。
- **Windows 便攜版**：release 資產新增 Windows portable zip（#106）。

## 修復

- **統計口徑不一致**（根因）：隱私過濾前移到聚合入口 `get_daily_stats_with_segments_filtered`，總時長／應用／分類／小時分布／網站統計從此同口徑（概覽、MCP、日報生成、舊報重渲染統一走前置過濾，替換各處後置過濾）。
- **每小時分布可能超過 60 分鐘**：跨小時活動改為按小時切片、重疊區間只計一次（歸最新觀測），小時桶不再溢出。
- **切應用時長遺失**：上一應用無歷史記錄時也建立輕量回補記錄，不再丟棄 `duration_delta`。
- **MCP 讀介面未過濾隱私**：timeline／memory／sessions／current_context／weekly 資源統一套用「忽略應用／排除域名」過濾。
- **MCP notifications/initialized 回傳無效空響應**（#107）。
- **Linux CI apt 抖動**：azure 鏡像換回 archive.ubuntu.com 主源，`apt-get update` 加 180s 超時＋3 次退避重試（曾導致 v1.0.51 一次 6 小時的取消）。

## 優化

- **統一亮暗主題色彩**：淺色文字從 8 級 slate 收斂到 4 級；深色全面遷移到 GitHub Dark 調色盤（`#0d1117/#161b22/#21262d/#30363d/#484f58`），`text-white`→`#e6edf3`，所有陰影補 `dark:` 覆蓋；移除設定儲存塢漸層、關於頁更新橫幅置中、移除更新 toast。
- **語言切換展示**：語言按鈕顯示語言全稱（不再只是 ZH/EN/TW 短碼）、選中項加 ✓、順序統一為 ZH/EN/TW。
- **介面細節**：捲軸與卡片圓角細化、小時活躍度圖改 24 欄網格對齊（首尾刻度不再溢出）、無資料時摘要顯示佔位、深色白底高光清理、設定 chip 形狀、側欄導覽啟用態（#108）。
- **README 重寫**：三語（EN/ZH/TW）引言與特性文案重寫更準確，新增 Localhost API 措辭守護測試，群組資源壓縮。

## macOS 安裝說明

首次開啟可能提示「已損壞，無法開啟」，請執行：

```bash
sudo xattr -rd com.apple.quarantine '/Applications/Work Review.app'
```
