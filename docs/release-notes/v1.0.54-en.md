# Work Review v1.0.54

**Release date**: 2026-07-02

## Fixes

- **Hidden report blocks couldn't be restored** (#109): `apply_preferences` no longer filters hidden blocks at generation time; hiding is now purely a display-layer concern. Restoring a block in "Manage blocks" makes it visible immediately, no regeneration needed.
- **Tray menu showed Chinese in English mode** (#115): Added a `locale` field to the config; tray menu labels now have zh-CN / en / zh-TW mappings, and switching language in the sidebar refreshes the tray immediately.
- **Assistant replies didn't follow the question's language** (#116): All three paths (Agent / Direct / Fast) now reply in the language of the user's question — English question → English answer, Chinese question → Chinese answer, regardless of the language stored in the work records.
- **FastPath category names leaked Chinese**: `fast_answer` now branches category names by the question's language; English mode no longer mixes in Chinese category labels.
- **Updater fallback leak**: Unrecognized backend status strings are no longer returned as-is, preventing Chinese leakage in English mode.
- **Linux build failure** (1.0.53 leftover): Fixed `include_str!` path depth, cross-module references, and missing `PathBuf` import after the `commands.rs` split.
- **Tray command compilation**: `set_app_locale` made private (avoiding macro re-export conflict) + `AppHandle` ownership fix.
- **Report generation closure syntax**: Fixed a stray `>` in the `apply_preferences` closure parameter.

## Improvements

- **Silent-resident GPU usage** (#118): When the main window is hidden (silent autostart + lightweight mode), all CSS animations are now paused. WebView2 GPU usage drops from 7-30% to near zero.
- **Overview date bar**: Removed the separator line below the date bar (which read as an "unfinished loading" indicator) and tightened the height. The calendar-style date picker was tested but the original text input was kept.
- **Desktop pet settings copy**: Shortened and polished avatar descriptions; added an explicit "uses a small amount of tokens" note to the proactive-AI data notice.

## macOS install note

On first launch you may see a "damaged" warning. Run:

```bash
sudo xattr -rd com.apple.quarantine '/Applications/Work Review.app'
```
